How It Works

RuView turns commodity WiFi into a contactless sensor using Channel State Information (CSI) — the same radio reflections WiFi uses to optimize its own signal.

1

CSI Collection

$9 ESP32-S3 boards capture WiFi Channel State Information from ambient router signals

2

Phase Sanitization

Hardware noise is removed and signal phases are normalized across all 30 subcarriers

3

Signal Processing

Spatio-temporal analysis extracts motion, breathing, and heartbeat signals from RF perturbations

4

AI Inference

8 KB quantized neural network extracts pose keypoints, presence, and activity classification

5

Spatial Intelligence

Multi-node fusion enables through-wall tracking, vital signs, and environment mapping
